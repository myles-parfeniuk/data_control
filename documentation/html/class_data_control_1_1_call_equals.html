◆ set()

template<typename T >
void CallEquals::set ( T  new_data)

Set value of data.

Overwrite current data with a new value. When this function is called, if the new data matches the comparison data all followers within this DataWrapper object's follower list will have their call-backs executed. Upon completion of call-back executions, the current data value will be over-written with the data passed into this function. If the get() method is called within a callback it will return the current data, the input parameter of a call-back is the new data that is to overwrite the current data.

Parameters
new_datathe new value to over-write current data with.

void, nothing to return
