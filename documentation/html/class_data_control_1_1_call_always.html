Method responsible for executing non-immediate call-back functions of followers registered to this DataWrapper object. Executes follower callback always. This function is called from CbHelper's main task.

◆ set()

template<typename T >
void CallAlways::set ( T  new_data)

Set value of data.

Overwrite current data with a new value. When this function is called, any followers registered to this DataWrapper object will have their call-backs executed. Upon completion of call-back executions, the current data value will be over-written with the data passed into this function.
