void Call\+Same\+::cb\+\_\+task\hspace{0.3cm}{\ttfamily [private]}}



Call-\/back task. 

Task responsible for executing the call-\/back functions of followers registered to this \doxylink{class_data_control_1_1_data_wrapper}{Data\+Wrapper} object. Executes follower callback if and only if the new data is the same as the previous data. This task is started when \doxylink{class_data_control_1_1_call_same_a73b1902bf71e964b96bcdb19ff4a4804}{set()} is called, and self-\/deletes upon completion.
