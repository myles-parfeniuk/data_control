\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename T$>$\newline
class Data\+Wrapper$<$ T $>$}
Base \doxylink{class_data_wrapper}{Data\+Wrapper} class. 

Stores a variable/data and allows for execution of callback functions when it is changed. When changing the data is desired, the set() method should be called, upon which any callbacks registered with the \doxylink{class_data_wrapper_a136bec7b06ed572db651ad3c416e4108}{follow()} method will be executed, and the data updated. \doxylink{class_data_wrapper}{Data\+Wrapper} objects should not be instantiated directly, instead insatiate objects from derived classes of which definitions can be found in cb\+\_\+types/ directory. ~\newline

Follow \doxylink{class_data_wrapper}{Data\+Wrapper} object. 

Trigger callback when this \doxylink{class_data_wrapper}{Data\+Wrapper} object is set. The call-\/back function parameter should have a single input parameter of type T. This input will be the new value of the data being set. Calling the \doxylink{class_data_wrapper_a39f4fc58c4506c8b2c0b589d951d91d4}{get()} method from within the callback will return the current value. When all callbacks have been executed the current value will over written with the new value. ~\newline
