Set value of data. 

Overwrite current data with a new value. When this function is called, if the new data matches the comparison data all followers within this \doxylink{class_data_control_1_1_data_wrapper}{Data\+Wrapper} object\textquotesingle{}s follower list will have their call-\/backs executed. Upon completion of call-\/back executions, the current data value will be over-\/written with the data passed into this function. If the \doxylink{class_data_control_1_1_data_wrapper_aee43e40ca8569976ee7ef896760df181}{get()} method is called within a callback it will return the current data, the input parameter of a call-\/back is the new data that is to overwrite the current data.
